# 工业行业特性详解

## 一、薄膜/塑料行业

### 1.1 米重换算公式
薄膜行业核心计算: **重量 = 长度 × 幅宽 × 厚度 × 密度**
```
weight_kg = length_m × width_m × thickness_mm × density_g_cm3 / 1000
```

### 1.2 典型业务场景
| 业务 | 计量单位 | 库存汇总 | 单据展示 |
|---|---|---|---|
| 采购入库 | 吨/卷/米 | 主单位(卷) | 详细显示吨+米 |
| 销售出库 | 卷/米/公斤 | 主单位(卷) | 按客户习惯切换 |
| 盘点 | 主单位 | 主单位 | 显示辅助单位 |

### 1.3 分切/复卷/裁切
- **分切**: 大卷 -> 多小卷
  - 入: 1 卷 BOPP 20um×1000mm×4000m
  - 出: 4 卷 BOPP 20um×250mm×4000m
  - 损耗: 切边 ≈ 0.5%

- **复卷**: 散装 -> 整齐成品卷
  - 入: 100 米 BOPP 半成品
  - 出: 1 卷 BOPP 100m

- **裁切**: 按客户尺寸定制
  - 入: 1 张 2000×3000mm PE 板
  - 出: 10 张 1000×600mm PE 板

## 二、塑料行业 (注塑/吹塑/挤出)

### 2.1 单位体系
- 颗粒: 千克(kg)、吨(t)
- 制品: 个、件
- 模具: 套

### 2.2 BOM 特性
- 主料 (PE/PP/ABS 颗粒): 90%
- 辅料 (色母/添加剂): 5%
- 包材: 5%
- 损耗率: 2-5%

### 2.3 成本归集
```
成本 = 主料成本 + 辅料成本 + 包装成本 + 人工 + 制造费用
     = SUM(实际用量 × 单价) + 工序成本
```

## 三、五金行业

### 3.1 单位
- 件、套、把、只
- 长度: 毫米、米
- 重量: 克、千克

### 3.2 BOM 特性
- 主件: 1
- 标准件 (螺钉/螺母): N 个, 来自标准件库
- 表面处理 (电镀/喷涂): 委外
- 损耗率: 0.5-1%

## 四、加工行业 (委外/分切)

### 4.1 委外发料
- 原料发给外协厂
- 加工费按件/按重量结算
- 收回应收加工费单

### 4.2 分切
- 原料出库 -> 委外
- 成品入库 -> 委外
- 加工费入库时确认

## 五、工贸一体企业

### 5.1 多模式支持
- 采购销售 (贸易)
- 加工生产 (制造)
- 委外加工 (外协)

### 5.2 单据流转
```
销售订单 (客户)
  -> 库存不足 -> 采购订单 (供应商) OR 生产加工单 (内部)
  -> 采购入库 OR 成品入库
  -> 销售出库 (毛利 = 售价 - 成本)
```

## 七、性能优化 (工业大数据量)

### 7.1 库存表分区
```sql
ALTER TABLE inv_ledger PARTITION BY RANGE (YEAR(biz_date) * 100 + MONTH(biz_date)) (
  PARTITION p202501 VALUES LESS THAN (202502),
  PARTITION p202502 VALUES LESS THAN (202503),
  ...
);
```

### 7.2 报表数据仓库
- 每天凌晨同步销售汇总到 `rpt_daily_snapshot`
- 复杂报表走 `clickhouse` / `StarRocks`

### 7.3 索引
- `inv_stock (warehouse_id, product_id, batch_no)` 联合索引
- `inv_ledger (biz_date, product_id)` 范围查询
- `sal_delivery (customer_id, bill_date)` 客户分析

## 八、扩展建议
