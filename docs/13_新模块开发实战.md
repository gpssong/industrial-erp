# 新模块开发实战

> 适用人员: 后端 / 前端 / 全栈开发
> 阅读前提: 已阅读 10_后端开发指南.md、11_前端开发指南.md
> 本文档以「**运输管理(Transport)**」模块为示例,从零走通「SQL → Entity → Mapper → Service → Controller → 前端页面 → 菜单权限」完整链路。完成后该模块即可在 PC 后台、桌面端、App 中使用。

## 一、需求拆解

我们要新增「运输管理」模块,功能如下:

| 字段 | 描述 |
|---|---|
| **业务对象** | 运单(车辆、司机、客户、关联销售出库单、运费、状态) |
| **单据状态** | 草稿 / 已发车 / 已签收 / 已结算 / 已取消 |
| **关联单据** | 销售出库单(SalDelivery) |
| **权限点** | 运输单:新增、修改、删除、查询、审核、结算、打印 |
| **报表** | 运费汇总、按车辆/司机/客户统计 |
| **行业特性** | 支持按重量/体积计费,支持回单上传(图片) |

## 二、整体流程

新增一个完整模块,标准 12 步:

1. 数据库建表 SQL
2. 后端 Entity 实体类
3. 后端 Mapper 接口与 XML
4. 后端 Service 业务层
5. 后端 Controller 接口层
6. 后端 DTO / VO
7. 菜单与权限种子数据
8. 前端 API 模块
9. 前端列表页 + 表单页
10. 前端路由注册
11. 前后端联调测试
12. 打包发布

预计工作量:1 个熟练开发 1.5 ~ 2 个工作日。

**关键设计点说明**:
- 软删除字段 `deleted TINYINT DEFAULT 0` —— 所有表都加。
- 唯一索引必须带 `deleted` 列,否则软删除与新增会冲突。
- 冗余字段(`vehicle_no`、`driver_name` 等)避免 JOIN,列表查询性能更优;更新主表时同步更新冗余字段。
- `tenant_id` 预留多租户。
- 索引覆盖主要查询场景:日期、状态、关联单据、外键。

## 六、第 4 步:Service 业务层

> **务必先阅读 `StockService` 的实现**(`modules/inventory/service/StockService.java`),本模块不涉及库存,但所有写操作都应走「**事务 + 校验 + 主从表 + 状态机**」通用模式。

**TraOrderService.java**

```java
package com.industrial.erp.modules.transport.service;

import com.industrial.erp.common.R;
import com.industrial.erp.exception.BizException;
import com.industrial.erp.modules.transport.dto.TraOrderSaveDTO;
import com.industrial.erp.modules.transport.dto.TraOrderQueryDTO;
import com.industrial.erp.modules.transport.entity.TraOrder;
import com.industrial.erp.modules.transport.entity.TraOrderDetail;
import com.industrial.erp.modules.transport.mapper.TraOrderDetailMapper;
import com.industrial.erp.modules.transport.mapper.TraOrderMapper;
import com.industrial.erp.security.SecurityContext;
import com.industrial.erp.utils.BillNoGenerator;
import com.baomidou.mybatisplus.core.metadata.IPage;
import com.baomidou.mybatisplus.extension.plugins.pagination.Page;
import lombok.RequiredArgsConstructor;
import org.springframework.stereotype.Service;
import org.springframework.transaction.annotation.Transactional;

import java.math.BigDecimal;
import java.time.LocalDate;
import java.util.List;

/**
 * 运输单服务
 */
@Service
@RequiredArgsConstructor
public class TraOrderService {

    private final TraOrderMapper orderMapper;
    private final TraOrderDetailMapper detailMapper;
    private final BillNoGenerator billNoGenerator;

    /**
     * 分页查询
     */
    public IPage<TraOrder> page(TraOrderQueryDTO q) {
        return orderMapper.pageList(new Page<>(q.getPageNum(), q.getPageSize()), q);
    }

    /**
     * 详情
     */
    public TraOrder detail(Long id) {
        TraOrder o = orderMapper.selectFullById(id);
        if (o == null) throw BizException.of("运输单不存在");
        o.setDetails(detailMapper.selectByOrderId(id));
        return o;
    }

    /**
     * 新增(草稿)
     */
    @Transactional(rollbackFor = Exception.class)
    public TraOrder save(TraOrderSaveDTO dto) {
        // 1. 校验
        validate(dto);
        // 2. 构造主表
        TraOrder o = new TraOrder();
        o.setBillNo(billNoGenerator.next("TS"));
        o.setBillDate(dto.getBillDate() == null ? LocalDate.now() : dto.getBillDate());
        o.setVehicleId(dto.getVehicleId());
        o.setVehicleNo(dto.getVehicleNo());
        o.setDriverId(dto.getDriverId());
        o.setDriverName(dto.getDriverName());
        o.setDriverPhone(dto.getDriverPhone());
        o.setCustomerId(dto.getCustomerId());
        o.setCustomerName(dto.getCustomerName());
        o.setDeliveryId(dto.getDeliveryId());
        o.setDeliveryNo(dto.getDeliveryNo());
        o.setStartAddr(dto.getStartAddr());
        o.setEndAddr(dto.getEndAddr());
        o.setDistanceKm(dto.getDistanceKm() == null ? BigDecimal.ZERO : dto.getDistanceKm());
        o.setFreightAmount(dto.getFreightAmount() == null ? BigDecimal.ZERO : dto.getFreightAmount());
        o.setTotalWeight(dto.getTotalWeight());
        o.setTotalVolume(dto.getTotalVolume());
        o.setStatus(0); // 草稿
        o.setSettleStatus(0);
        o.setRemark(dto.getRemark());
        o.setCreatedBy(SecurityContext.getUserId());
        o.setCreatedByName(SecurityContext.getUserName());
        orderMapper.insert(o);
        // 3. 写明细
        saveDetails(o.getId(), dto.getDetails());
        return o;
    }

    /**
     * 修改
     */
    @Transactional(rollbackFor = Exception.class)
    public void update(TraOrderSaveDTO dto) {
        TraOrder old = orderMapper.selectById(dto.getId());
        if (old == null) throw BizException.of("运输单不存在");
        if (old.getStatus() != 0) throw BizException.of("仅草稿状态可修改");
        // ... 字段赋值
        orderMapper.updateById(old);
        // 删旧明细
        detailMapper.deleteByOrderId(old.getId());
        saveDetails(old.getId(), dto.getDetails());
    }

    /**
     * 删除
     */
    @Transactional(rollbackFor = Exception.class)
    public void delete(Long id) {
        TraOrder o = orderMapper.selectById(id);
        if (o == null) throw BizException.of("运输单不存在");
        if (o.getStatus() != 0) throw BizException.of("仅草稿状态可删除");
        orderMapper.deleteById(id);
        detailMapper.deleteByOrderId(id);
    }

    /**
     * 状态机: 发车
     */
    @Transactional(rollbackFor = Exception.class)
    public void depart(Long id) {
        TraOrder o = orderMapper.selectById(id);
        if (o == null) throw BizException.of("运输单不存在");
        if (o.getStatus() != 0) throw BizException.of("仅草稿状态可发车");
        o.setStatus(1);
        o.setDepartTime(java.time.LocalDateTime.now());
        orderMapper.updateById(o);
    }

    /**
     * 状态机: 签收
     */
    @Transactional(rollbackFor = Exception.class)
    public void sign(Long id, String signImage) {
        TraOrder o = orderMapper.selectById(id);
        if (o == null) throw BizException.of("运输单不存在");
        if (o.getStatus() != 1) throw BizException.of("仅已发车状态可签收");
        o.setStatus(2);
        o.setSignTime(java.time.LocalDateTime.now());
        o.setSignImage(signImage);
        orderMapper.updateById(o);
    }

    /**
     * 状态机: 取消
     */
    @Transactional(rollbackFor = Exception.class)
    public void cancel(Long id, String reason) {
        TraOrder o = orderMapper.selectById(id);
        if (o == null) throw BizException.of("运输单不存在");
        if (o.getStatus() == 2) throw BizException.of("已签收不可取消");
        o.setStatus(9);
        o.setRemark((o.getRemark() == null ? "" : o.getRemark()) + " [取消] " + reason);
        orderMapper.updateById(o);
    }

    // ---------- 私有方法 ----------
    private void validate(TraOrderSaveDTO dto) {
        if (dto.getVehicleId() == null) throw BizException.of("请选择车辆");
        if (dto.getDriverId() == null) throw BizException.of("请选择司机");
        if (dto.getDetails() == null || dto.getDetails().isEmpty()) {
            throw BizException.of("至少添加一条货物明细");
        }
    }

    private void saveDetails(Long orderId, List<TraOrderDetail> details) {
        if (details == null) return;
        for (TraOrderDetail d : details) {
            d.setId(null);
            d.setOrderId(orderId);
            detailMapper.insert(d);
        }
    }
}
```

## 十五、常见错误与检查清单

| 错误 | 原因 | 排查 |
|---|---|---|
| 前端菜单不显示 | 菜单种子未导入 / 前端缓存 | 重启后端、清浏览器缓存、检查 Redis 菜单缓存 |
| 接口 403 | 权限码未配 / 用户未授权 | 检查 `sys_role_menu` 是否有该菜单 |
| 接口 401 | Token 过期 | 重新登录 |
| 单据号重复 | `BillNoGenerator` Redis 异常 | 检查 Redis 连接,Redis 不可用会降级到 DB 序列 |
| MyBatis Plus 找不到字段 | 表名/字段名不一致 | 开启 MP `mapUnderscoreToCamelCase` (默认开) |
| XML 中文注释乱码 | 文件非 UTF-8 | IDE 改编码为 UTF-8 |
| 软删除数据查询不到 | 默认过滤 `deleted=0` | 用 `Wrapper` 显式查询或后台管理里查 |
| 事务不生效 | 自调用 / 非 public 方法 | 加 `@Transactional` 在 Service 上,Controller 调 Service |
