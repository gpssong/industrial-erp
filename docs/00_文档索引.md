## 🚀 NAS 部署指南对照

| 平台 | 文档 | 推荐度 | 备注 |
|---|---|---|---|
| **飞牛 OS (fnOS)** | `18_飞牛OS部署指南.md` | ⭐⭐⭐⭐⭐ | **首选**, 免费无授权, 原生 Compose V2 无镜像兼容问题, btrfs 快照 |
| **群晖 DS918+ (DSM 7.2+)** | `17_群晖NAS部署指南.md` | ⭐⭐⭐⭐ | 已有群晖机器可选, 需注意 Container Manager 镜像标签兼容性 |
| **通用 Linux 服务器** | `02_部署方案.md` | ⭐⭐⭐ | CentOS / Ubuntu / Debian 通用流程 |

## 🛠 飞牛 OS 一键脚本

放在 `scripts/` 目录下:

| 脚本 | 用途 |
|---|---|
| `scripts/fnos-deploy.sh` | 一键部署 (检查环境、生成 .env、btrfs 快照目录、构建启动) |
| `scripts/fnos-backup.sh` | 一键备份 (数据库 dump + btrfs 快照 + 自动清理 30 天前数据) |
| `scripts/fnos-upgrade.sh` | 一键升级 (备份 → git pull → 重新构建) |
