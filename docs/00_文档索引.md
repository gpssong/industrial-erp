# 企业级工业 ERP - 文档索引

> 本目录包含项目的全部开发与运维文档。文档已与 **当前代码 v1.0** 完全对齐,所有接口、表、模块都可在代码中检索到对应实现。

## 📚 文档结构

```
docs/
├── 00_文档索引.md              ← 你正在看的文件
├── 01-开发文档(总纲).md         ← 项目总纲,架构/技术栈/业务全景 (最重要,先读)
├── 02-数据库设计.md             ← ER 图 / 表结构 / 索引策略
├── 03-API接口设计规范.md         ← RESTful 规范 / 统一返回 / 鉴权
│
├── 01_业务逻辑说明.md           ← 库存/成本/生产/往来 核心业务详解
├── 02_部署方案.md               ← Docker / Nginx / 监控 / 升级
├── 03_API接口文档.md            ← 全部 REST 接口列表
├── 04_行业特性.md               ← 薄膜/塑料/五金/加工 行业适配
│
├── 10_后端开发指南.md           ← 新增模块 / 代码规范 / 单元测试
├── 11_前端开发指南.md           ← Vue3 组件模式 / API 封装 / 状态管理
├── 12_本地开发环境搭建.md       ← 从零搭建开发环境 (Mac/Linux/Windows)
├── 13_新模块开发实战.md         ← 完整示例: 一步步新增一个业务模块
├── 14_测试与调试指南.md         ← 单元测试 / 接口测试 / 常见 Bug
├── 15_Git与协作规范.md          ← 分支策略 / 提交规范 / Code Review
└── 16_常见问题FAQ.md            ← 部署 / 启动 / 性能 / 业务问题
```

## 🎯 推荐阅读路径

### 你是新加入的开发者
1. `01-开发文档(总纲).md` - 了解整体架构
2. `12_本地开发环境搭建.md` - 搭建开发环境
3. `10_后端开发指南.md` (后端) 或 `11_前端开发指南.md` (前端)
4. `13_新模块开发实战.md` - 跟着示例动手

## 🔄 维护规则

- 修改任何业务逻辑、SQL、接口签名,**必须同步更新对应文档**
- 文档与代码不一致以代码为准,但必须立即修订文档
- 新增模块必须补充: 接口文档、数据库设计、开发指南示例
